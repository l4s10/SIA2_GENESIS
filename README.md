# Sistema integrado de administración (SIA V2)

Este proyecto de tesis fue realizado para la VIII dirección regional del Servicio de Impuestos Internos en Chile.

# Tecnologías utilizadas

- Laravel
- XAMPP (Apache y MySQL)
- Composer
